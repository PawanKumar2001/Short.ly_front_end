# Short.ly - A URL Shortner

This is the front - end for Short.ly web application. 

## Vercel Link
Click This link to see Short.ly in action - https://shortlyfrontend.vercel.app/

# Technology Used

The Front - end of Short.ly is based on 
a. React + Vite
b. Tailwind CSS

## Features
1. User Authentication
2. URL Shortning
3. URL History
4. Copy to Clipboard
5. Dark/Light Mode
6. Responsive UI
